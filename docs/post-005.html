放課後等デイサービスの利用料金の仕組みと自己負担上限月額

はじめに――「料金が心配」という方へ

放課後等デイサービスや児童発達支援を利用したいと思っても、「費用はどのくらいかかるの？」と不安になる保護者の方は多くいらっしゃいます。実は、これらのサービスは障害児通所支援として国の制度に基づく給付が行われるため、かかった費用のほとんどを国と自治体が負担してくれます。自己負担は原則として1割にとどまり、さらに所得に応じた「上限月額」が設けられているため、家計への影響を一定の範囲に抑えられる仕組みになっています。

この記事では、利用料金の基本的な仕組みから自己負担上限月額の区分・手続きの流れまで、できるだけわかりやすく説明します。なお、金額や制度の詳細は自治体や事業所によって異なる場合があります。必ず通所を検討している事業所や、お住まいの市区町村窓口にご確認ください。

利用料金の基本的な仕組み

1. サービスの費用はどうやって決まるの？

放課後等デイサービス・児童発達支援のサービス費用は、国が定めた「報酬単価」をもとに計算されます。報酬単価は支援の内容や時間、事業所の体制などによって異なり、1回あたりの費用（1単位＝約10円）として算定されます。この費用の9割を国・都道府県・市区町村が負担し、残りの1割が保護者の自己負担となります。

2. 自己負担上限月額とは？

自己負担が1割といっても、利用日数が多い場合は月々の負担が大きくなってしまいます。そこで設けられているのが「自己負担上限月額」です。1か月に支払う自己負担額が上限額を超えた場合、超過分は支払わなくてよい仕組みになっています。つまり、どれだけ多くサービスを利用しても、上限額以上を支払う必要はありません。

所得区分ごとの自己負担上限月額

自己負担上限月額は、保護者（世帯）の所得によって以下の4区分に分けられています。なお、下記の金額は国が定めた基本的な目安です。自治体によって独自の助成制度がある場合もあるため、お住まいの市区町村に確認することをおすすめします。

  • 生活保護受給世帯・低所得世帯（市民税非課税世帯）：月額 0円（自己負担なし）
  • 一般1（市民税課税世帯のうち、市民税所得割額が28万円未満）：月額 4,600円
  • 一般2（市民税所得割額が28万円以上の世帯）：月額 37,200円
  • 一般3（入所施設利用者などの特例）：別途規定あり（詳細は自治体へ）

多くのご家庭は「一般1」に該当し、月4,600円が上限となります。所得割額の計算方法は自治体によって異なる場合があるため、正確な区分は市区町村窓口または事業所にご相談ください。

「管理事業所」による上限額管理の仕組み

複数の事業所を利用する場合、自己負担上限月額は利用する事業所全体を合算して管理されます。このとき、いずれか1か所の事業所が「上限額管理事業所」となり、月ごとの支払い合計が上限額を超えないよう調整してくれます。利用している事業所が複数ある場合は、どの事業所が管理事業所になるかをあらかじめ確認しておきましょう。

実費負担（自己負担上限額とは別にかかる費用）

上限月額で管理されるのはサービス費用の1割分のみです。以下のような実費は別途発生する場合があります。事業所によって異なるため、契約前に必ず確認しましょう。

  • おやつ・食事代
  • 活動材料費・教材費
  • 送迎にかかる費用（事業所により無料の場合も）
  • 行事・外出時の入場料など

通所開始前に確認したいこと

  1. 受給者証の「上限月額」欄を確認する：受給者証には所得区分に応じた上限月額が記載されています。
  2. 事業所の重要事項説明書を読む：サービス費の1割分のほか、実費負担の内訳が記載されています。
  3. 複数の事業所を利用する場合は上限管理事業所を確認する：月ごとの精算がスムーズになります。
  4. 自治体の独自助成制度を調べる：自己負担をさらに軽減できる制度がある自治体もあります。

まとめ

放課後等デイサービス・児童発達支援の利用料金は、国の給付制度により自己負担が原則1割に抑えられ、さらに所得区分に応じた上限月額が設定されているため、多くのご家庭で毎月の負担は月額4,600円以下になります。ただし、実費部分は事業所ごとに異なるため、契約前に必ず確認することが大切です。

「自分の家庭はどの区分になる？」「受給者証はどう申請すればいい？」など、不明点はお住まいの市区町村の障害福祉担当窓口や、検討中の事業所のスタッフに遠慮なく相談してみてください。専門のスタッフが丁寧にサポートしてくれます。

よくある質問

自己負担上限月額はどうやって決まりますか？

保護者世帯の市民税所得割額をもとに区分が決まります。非課税世帯は0円、課税世帯は所得割額に応じて月4,600円または37,200円となります。正確な区分はお住まいの市区町村窓口にご確認ください。

複数の事業所を利用した場合、上限額はそれぞれにかかりますか？

いいえ、複数の事業所を利用しても上限額は合算で管理されます。「上限額管理事業所」が各事業所の利用額を合計し、上限を超えた分は請求されない仕組みです。

おやつ代や送迎費も上限額に含まれますか？

おやつ代・教材費・送迎費などの実費は上限月額の管理対象外です。これらはサービス費の1割とは別に事業所ごとに設定されているため、契約前に必ず確認しましょう。

受給者証がなくてもサービスを利用できますか？

放課後等デイサービス・児童発達支援を利用するには受給者証が必要です。お住まいの市区町村の障害福祉担当窓口で申請できます。申請から交付まで日数がかかる場合があるため、早めに相談することをおすすめします。
